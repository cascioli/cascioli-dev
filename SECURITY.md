# Security Policy

## Scope

This policy covers the cascioli.dev website and its serverless API endpoints
(`/api/osint/scan`, `/api/contact/contact`).

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report privately via email: **<security@simonecascioli.it>**

Include:

- Description of the vulnerability and potential impact
- Steps to reproduce
- Affected endpoint or component

You can expect an acknowledgement within **48 hours** and a status update within
**7 days**.

## Out of Scope

- Attacks requiring physical access
- Social engineering
- Denial-of-service attacks
- Findings from automated scanners without proof of exploitability
- Rate limiting / brute-force on the contact form

## Disclosure

Responsible disclosure is appreciated. Please allow reasonable time to patch
before any public disclosure.
